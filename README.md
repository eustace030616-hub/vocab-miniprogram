# Vocab Miniprogram

a vocabulary testing miniprogram with CEFR leveling system.

## features

- In-test dynamic difficulty 
- CEFR word bank with weighted word selection
- Quick tes & infinite mode

## tech stack

- WeChat mini program (WXML/WXSS/JS)
- WeChat Dev Tool
- Claude Code + MiniMax 2.7

## future improvement

- better estimation accuracy
- better word bank
- infinite mode implementation
- engligh->german vocabulary estimation
